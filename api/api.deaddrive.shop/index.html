API Documentation - File Upload Service

This API allows users to upload files from Google Drive to external services (Filepress and Hydrax) using a provided API key.

Endpoint

GET https://api.[DEADDRIVE_DOMAIN]/upload

Parameters

The API accepts the following query parameters:

  • api (required): The user's API key for authentication and authorization.
  • drive_id (required): The Google Drive file ID of the file to upload.

Usage

Make a GET request to the endpoint with the required parameters. Example:

GET https://api.[DEADDRIVE_DOMAIN]/upload?api=your_api_key&drive_id=your_drive_file_id

Response

The API returns a JSON response with the following structure:

Success Response

On successful upload or if the file already exists, the response includes file details and links:


    {
        "status": "success",
        "fileStatus" : "NewAdd" or "AlreadyExist",
        "name": "file_name.ext",
        "key": "deaddrive_file_id",
        "driveId": "google_drive_file_id",
        "size": size
        "download": "https://deaddrive_domain/file/unique_file_uid",
        "watch": "https://deaddrive_domain/embed/unique_file_uid"
    }

Note: If the file already exists, then "fileStatus": "AlreadyExist" field will be included in the response and Links of File will be Updated.

Error Response

If an error occurs, the response includes an error message:

{
    "status": "error",
    "message": "Error message describing the issue"
}

Possible Error Messages

  • No API provided: The api parameter is missing.
  • No Google Drive File ID provided: The drive_id parameter is missing.
  • Not a valid API key: The provided API key is invalid or not found.
  • Not a valid Google Drive ID: The Google Drive file ID is invalid or the file does not exist.

Example Request

GET https://api.deaddrive.shop/upload?api=abc123&drive_id=1A2B3C4D5E6F7G8H9I0J

Example Success Response

{
    "status": "success",
    "fileStatus" : "NewAdd",
    "name": "example.mp4",
    "key": "xyz789",
    "driveId": "google_drive_file_id",
    "size": size
    "download": "https://deaddrive_domain/file/xyz789",
    "watch": "https://deaddrive_domain/embed/xyz789"
}

Example Success Response (File Exists)

{
    "status": "success",
    "fileStatus" : AlreadyExist",
    "name": "example.mp4",
    "key": "xyz789",
    "driveId": "google_drive_file_id",
    "size": size
    "download": "https://deaddrive_domain/file/xyz789",
    "watch": "https://deaddrive_domain/embed/xyz789",
}

Example Error Response

{
    "status": "error",
    "message": "Not a valid Google Drive ID"
}
